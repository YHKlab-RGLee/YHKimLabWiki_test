## 1. 왜 Linux를 배우는가

연구실 계산 환경은 대부분 Linux 서버(HPC)를 중심으로 동작합니다.

- 계산 도구/컴파일러/스크립트가 Linux 기준으로 제공되는 경우가 많음
- SSH 기반 원격 접속이 표준
- 반복 작업 자동화(bash script)와 대량 배치 실행에 유리

즉, Linux는 단순한 운영체제가 아니라 **연구 계산 실무의 공통 언어**에 가깝습니다.

---

## 2. HPC vs 개인 PC

### 2.1 개인 PC의 강점

- 즉시 사용 가능
- GUI 중심 작업이 편함
- 파일 접근이 직관적

### 2.2 HPC의 강점

- 대규모 CPU/메모리 자원 활용
- 장시간 계산 안정성
- 공용 실습 환경(패키지/설정 통일)

### 2.3 실습에서의 역할 분담

```text
로컬 PC: 편집/확인/보고서 작성
     |
     | SSH
     v
HPC: 계산 실행/로그 확인/결과 저장
```

---

## 3. Linux에서 자주 보는 요소

- **Shell**: 명령어를 입력하는 인터페이스(`bash` 등)
- **Path**: 파일 위치(`/home/...`)
- **Permission**: 파일 접근 권한(`rwx`)
- **환경 변수**: 실행 경로/설정값(`PATH`, `DISPLAY` 등)
